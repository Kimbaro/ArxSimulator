# ArxSimulator
Scanner-Sensor, Gateway, PLC를 가상으로 생성하여 Hub, Broker들의 부하테스트를 목적으로 하는 시뮬레이터의 Front-end 화면 입니다.


## 구조
- Project
    - Device
        - Scanner
            - Sensor
        - Gateway
    - PLC


## 기능
1. Project 추가, 편집, 삭제, 시뮬레이터 동작
2. Scanner 추가, 편집, 삭제
3. Gateway 추가, 편집, 삭제
4. Sensor 추가, 편집, 삭제


## 실행방법
``` shell
yarn
yarn start
```


## 버전
1. Nodejs 16.14.2


## 라이브러리
1. Typescript
2. MUI
3. Styled-components
4. react-query
5. react-hook-form
6. react-router-dom v6
7. recoil
